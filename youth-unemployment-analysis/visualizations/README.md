# Visualizations Directory

This directory contains all visualizations generated from the youth unemployment analysis.

## Organization

- `charts/`: Static charts and graphs (PNG, SVG, PDF formats)
- `interactive/`: Interactive visualizations (HTML, JSON formats)
- `maps/`: Geographic visualizations and maps
- `reports/`: Presentation-ready graphics for reports

## File Naming Convention

- `global_[type]_[year].png`: Global-level visualizations
- `regional_[region]_[type]_[year].png`: Regional visualizations
- `country_[country]_[type]_[year].png`: Country-level visualizations
- `trend_[period]_[metric].png`: Trend analysis visualizations

## Key Visualizations

### Global Analysis
- Global youth unemployment trends over time
- Regional comparisons and rankings
- Decade-by-decade changes
- Distribution analysis

### Regional Analysis
- Regional performance heatmaps
- Sub-regional comparisons
- Geographic distributions

### Country Analysis
- Country trajectory plots
- Performance rankings
- Correlation analysis

### Statistical Analysis
- Correlation matrices
- Time series decompositions
- Forecasting results

## Interactive Features

Several visualizations include interactive elements:
- Hover tooltips with detailed information
- Zoom and pan capabilities
- Filter options for different views
- Download options for static versions

## Usage

Visualizations are automatically generated by the analysis notebooks and saved in this directory. They are also embedded in the final report and presentation materials.
