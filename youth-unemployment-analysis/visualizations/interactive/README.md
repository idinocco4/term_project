# Interactive Visualizations Directory

This directory contains interactive visualizations and dashboards for dynamic data exploration.

## File Types

- HTML files with embedded Plotly.js visualizations
- Interactive dashboards with user controls
- JSON data files for web-based visualizations
- Standalone web applications

## Features

### Interactivity
- Hover tooltips with detailed information
- Zoom and pan capabilities
- Filter options for different views
- Clickable legends and data points

### Export Options
- Download static versions (PNG, SVG, PDF)
- Save filtered datasets
- Generate reports from current view

## Usage

Interactive visualizations are created by the dashboard notebook (Notebook 6) and provide enhanced exploration capabilities beyond static charts.

## Technical Details

- Built with Plotly.js for web compatibility
- Responsive design for different screen sizes
- Optimized for modern web browsers
- Can be embedded in websites or shared via links
