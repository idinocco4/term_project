# Charts Directory

This directory contains static charts and graphs generated from the youth unemployment analysis.

## File Naming Convention

- `global_[type]_[year].png`: Global-level visualizations
- `regional_[region]_[type]_[year].png`: Regional visualizations
- `country_[country]_[type]_[year].png`: Country-level visualizations
- `trend_[period]_[metric].png`: Trend analysis visualizations

## Chart Types

### Global Analysis
- Time series plots
- Trend analysis charts
- Distribution histograms
- Summary statistics plots

### Regional Analysis
- Regional comparison bar charts
- Geographic heatmaps
- Sub-regional breakdowns

### Country Analysis
- Country trajectory plots
- Performance ranking charts
- Comparative analysis plots

## Usage

Charts are automatically generated by the analysis notebooks and saved in this directory with descriptive filenames. They are also embedded in the final report and presentation materials.

## Supported Formats

- PNG (default for web display)
- SVG (vector format for high-quality printing)
- PDF (print-ready format)
- JPEG (compressed format for web)
