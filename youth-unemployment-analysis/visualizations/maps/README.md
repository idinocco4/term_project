# Maps Directory

This directory contains geographic visualizations and maps showing spatial patterns in youth unemployment.

## Map Types

### Choropleth Maps
- Country-level unemployment rates
- Regional aggregations
- Time-series animations
- Interactive hover details

### Bubble Maps
- Scaled by population or unemployment rates
- Multi-variable representations
- Temporal progression views

### Regional Maps
- Sub-national divisions
- Economic region boundaries
- Administrative unit analysis

## Data Sources

- Country boundary shapefiles
- Regional aggregation definitions
- Geographic coordinate systems
- Administrative division data

## Usage

Maps are generated using geographic data libraries and provide spatial context for the analysis. They help identify geographic patterns and regional disparities.

## Technical Notes

- Uses coordinate reference systems (CRS) appropriate for global mapping
- Handles missing geographic data gracefully
- Optimized for web display and printing
